# 前端 API

适用对象：正在查基座内建组件、微前端 SDK 和前端运行时能力的开发者。

## 从哪里开始查

- 想知道微前端应用、布局组件和单组件怎么初始化：看 [SDK 初始化方法](./sdk/initialization-methods/)
- 想查 SDK 在运行时注入了哪些工具方法：看 [SDK 自动补全方法](./sdk/completion-methods/)
- 想查平台基座已经内建了哪些组件：看 [基座内建组件](./buildin-comps/)

## 常见判断

- 你在“UI Schema 还是微前端”之间做判断时，先回到 [前端问题 FAQ](../../faq/frontend/)
- 你在“微前端、布局组件、登录页、组件库”之间做架构选择时，先回到 [前端融合开发方式](../../fusion-development/frontend-extension-paths)
- 你要查的是接入步骤而不是接口细节时，先看 [扩展前端](../../guide/advance/extend-frontend/)

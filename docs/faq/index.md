# 常见问题

这里收录潮汐栈使用过程中的高频判断题和排障题，优先覆盖“为什么没生效”“应该走哪条路径”“遇到问题先查什么”这类问题。

## 适合先来这里的情况

- 你已经有一个明确症状，但还没判断出该去哪个专题页
- 你遇到的是“没生效”“没加载”“没显示”“不知道先查哪里”这类问题
- 你不想先通读整套文档，而是想快速找到排查入口

## 常见问题入口

| 当前问题                                                 | 推荐入口                  |
| -------------------------------------------------------- | ------------------------- |
| 页面、微前端、UI Schema 或登录页相关问题                 | [前端问题](./frontend/)   |
| 扩展包、Bean、脚本、运行时装配相关问题                   | [后端问题](./backend/)    |
| 安装、升级、发布、RabbitMQ / WebSocket / Docker API 排障 | [运维问题](./operations/) |

## 从哪里开始看

- [前端问题](./frontend/)：页面渲染、UI Schema、微前端、自定义前端扩展
- [后端问题](./backend/)：扩展包加载、Spring Bean、运行时集成
- [运维问题](./operations/)：安装、升级、发布、备份、RabbitMQ、WebSocket、Docker API

## 如果这里还没覆盖你的问题

- 先确认你的问题属于 [低代码开发](../low-code/)、[融合开发](../fusion-development/) 还是 [运维管理](../operations/)
- 需要查对象、方法或 DSL 时，请转到 [API 参考](../api/)
- 需要先统一术语时，请转到 [核心概念](../concepts/)
- 如果你想先盘点平台已有能力边界，请转到 [平台能力地图](../reference/platform-capability-map)

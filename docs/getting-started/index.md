# 快速上手

适用对象：第一次接触潮汐栈的平台使用者、业务开发人员、专业开发人员和实施同学。

这个入口的目标很简单：先理解平台是什么，再把环境跑起来，最后完成第一个最小可用功能。

## 建议阅读顺序

1. [平台介绍](../overview)
2. [角色与典型场景](./roles-and-scenarios)
3. [环境准备](./environment-setup)
4. [安装开发平台](./install-develop-platform)
5. [创建第一个应用](./create-first-application)
6. [完成第一个业务功能](./first-business-feature)

如果你当前目标只是先把体验环境跑起来，通常读到“安装开发平台”就够了；应用创建和业务功能可以放到后面再继续。

## 你会在这里看到什么

- 平台定位、适用场景与角色分工
- 首次安装和启动开发平台所需的基础步骤
- 第一个应用和第一个业务功能的推荐路径
- 需要查看更多部署细节时应跳去哪里

## 补充阅读

- 如果你需要更完整的手工部署与初始化细节，请继续看 [运维管理](../operations/)
- 如果你准备团队环境或需要更细的配置拆分，可以继续看 [安装与初始化](../operations/installation-and-initialization)
